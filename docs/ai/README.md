# AI Operations Notes

This directory holds operational instructions for automated coding agents and
maintainers working in this repository. These files are not the product README
or public Vexic documentation.

- `docs/ai/AGENTS.md` is the source of truth for agent workflow rules, settled
  architecture boundaries, and verification expectations.
- `docs/ai/CLAUDE.md` is Claude Code prompt context that points at the shared
  agent instructions.
- `docs/ai/CONTEXT.md` is a product-language glossary for agent and maintainer
  planning.
- `docs/ai/REVIEW.md` is review-agent calibration. It lives here rather than
  at the repository root, where only minimal pointer files (`AGENTS.md`,
  `CLAUDE.md`) reference the shared agent instructions.
