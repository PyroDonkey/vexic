# AI Operations Notes

This directory holds operational instructions for automated coding agents and
maintainers working in this repository. These files are not the product README
or public Vexic documentation.

The agent source of truth lives at the repository root:

- `AGENTS.md` (repo root) is the source of truth for agent workflow rules,
  settled architecture boundaries, and verification expectations.
- `CLAUDE.md` (repo root) is Claude Code prompt context that imports `AGENTS.md`.

This directory holds the supporting agent references:

- `docs/ai/CONTEXT.md` is a product-language glossary for agent and maintainer
  planning.
- `docs/ai/REVIEW.md` is review-agent calibration.
