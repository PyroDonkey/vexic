# Vexic

This npm package reserves the `@pyrodonkey/vexic` package name for Vexic.

The v0.1 implementation is the Python package in the repository below. The
`vexic` name on PyPI is reserved by a placeholder package; the implementation
itself is not published yet, so build and run it from source with `uv`.

Source: https://github.com/PyroDonkey/vexic
